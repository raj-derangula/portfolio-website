AI Outbound Automation System

End-to-end outbound sales infrastructure for legal tech.

Built a fully automated outbound sales system for a legal tech SaaS company. The system ingests a CSV of prospects, executes AI-driven cold calls, qualifies leads through dynamic conversations using prompt chaining, and automatically books demos — all without human involvement. Call scripts are A/B tested to continuously optimize lead conversion rates. All activity is logged to Google Sheets in real time for the sales team to monitor.

  • Zero human involvement — fully automated from prospect ingestion to booked demo
  • Prompt chaining drives dynamic AI conversation flows that adapt in real time during calls
  • A/B testing of call scripts identifies highest-converting messaging to optimize lead qualification
  • Calendly API integration auto-schedules demos post-qualification with no manual follow-up
  • Real-time Google Sheets logging gives the sales team live pipeline visibility
Stack: n8n, Retell AI, OpenAI API, Calendly API, Google Sheets API
Status: Live at SecureSplit

ProfitPilot — AI Inventory Tracker

Resellers waste time manually logging order details with no easy way to track profit margins across their inventory.

Built a full-stack Next.js app that lets resellers upload order screenshots and uses Google's Gemini API to automatically extract product details, costs, and sale prices. Tracks active inventory and archives sold items with live revenue and margin calculations.

  • AI extraction via Gemini API eliminates manual data entry from order screenshots
  • Tracks revenue, costs, and profit margins across entire inventory in real time
  • Deployed to Vercel with secure server-side API key handling
Stack: React, Next.js 15, Gemini API, Tailwind CSS, Vercel

Airlock

Focus interception for your phone. Enter your mission before you open your apps.

Airlock is a pre-launch consumer iOS app that intercepts you before you open distracting social apps. Unlike every existing app blocker which simply blocks access, Airlock prompts you to declare your mission first and then displays it persistently through iOS Dynamic Island Live Activities so it stays visible the entire session. Building an audience and waitlist publicly before writing a single line of code.

  • Mission-declaration prompt intercepts app launches before distracting social apps open
  • iOS Dynamic Island Live Activities keep your stated mission visible throughout the session
  • Building waitlist and audience publicly pre-launch — zero code written yet
Stack: Swift, iOS Live Activities, Dynamic Island
Status: Pre-launch, waitlist live
